About This System

AI-Based Satellite Anomaly Detection for Spacecraft Health Monitoring — Mini Project, BMSCE 2025-26.

🏛 Model Architecture

PatchTST — Patch Time Series Transformer. One model per telemetry channel. Trained only on normal data (unsupervised). Anomaly score = forecasting error.

📡 Dataset

NASA SMAP / MSL telemetry. 82 channels of multivariate spacecraft sensor data with labeled anomaly intervals. Available on Kaggle.

📈 Key Results
  • Avg AUC-ROC : 0.882
  • Avg F1 : 0.689
  • Avg Accuracy: 0.957
  • Best (M-6) : F1=0.920
